<Steps>
  <Step title="Create a Yuki API key">
    Through Yuki Web services, you can create an access code (API key) that allows external software applications like Ravical to access your Yuki data.

    Follow the instructions in this Yuki article to create your API key:\
    https://support.yuki.be/en/support/solutions/articles/80000787839-setting-up-web-services-in-the-portal

    <Warning>
      **Important:** Only a Yuki user with the Portal administrator role can create an API key. This key will only provide access to domains and administrations that the portal administrator has access to.
    </Warning>
  </Step>
  <Step title="Navigate to Connections">
    1. Log in to your Ravical Dashboard
    2. Click on **Connections** in the left sidebar
    3. Locate the Yuki tile in the integrations list
    4. Click **Connect** to begin the setup process

    ![Screenshot2025 11 13at08 54 52 Pn](/images/Screenshot2025-11-13at08.54.52.png)
  </Step>
  <Step title="Connect with API Key">
    1. Paste the API key you created earlier into the input field
    2. Click **Connect with API Key**
    3. The connection will now be established and Ravical will start gathering relevant information from Yuki
  </Step>
</Steps>

**Connection not working?**

Check if:

- You copied the correct API key (without extra spaces)
- You're logged in as a Portal administrator in Yuki
- You have access to the administration you want to connect
